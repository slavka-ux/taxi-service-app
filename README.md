## Про проєкт
SkyBook — це сучасний та інтуїтивний вебдодаток для швидкого пошуку та бронювання авіаквитків. Головна мета проєкту — зробити процес купівлі квитків максимально простим, прозорим та приємним, позбавивши користувачів від заплутаних інтерфейсів та прихованих комісій.

Додаток розроблено з використанням чистого JavaScript (Vanilla JS) на фронтенді та Node.js на бекенді, що забезпечує блискавичну швидкість роботи та мінімальне навантаження на браузер.

## Можливості для користувачів

### 🧑‍💼 Для пасажирів:
* **Розумний пошук рейсів:** Користувачі можуть шукати авіаквитки за містом відправлення, призначення, датою вильоту та кількістю пасажирів. Система миттєво фільтрує доступні рейси.
* **Гнучка фільтрація:** Сортування результатів за ціною, тривалістю польоту, кількістю пересадок або часом вильоту.
* **Детальна інформація про рейс:** Кожен рейс містить повну інформацію — номер рейсу, авіакомпанію, час у дорозі, кількість пересадок, тип літака.
* **Бронювання в один клік:** Обравши рейс, користувач може забронювати квиток, вказавши особисті дані та контактну інформацію.
* **Особистий кабінет:** Всі бронювання зберігаються в профілі користувача з можливістю перегляду, скасування або зміни.

### 👨‍✈️ Для адміністраторів:
* **Управління рейсами:** Додавання нових рейсів, редагування розкладу, цін та наявності місць.
* **Моніторинг бронювань:** Перегляд усіх активних та завершених бронювань.
* **Статистика:** Аналітика популярних напрямків, завантаженості рейсів та доходів.

## Дизайн та інтерфейс
Візуальна складова SkyBook базується на принципах мінімалізму та простору. Використано світлу тему оформлення з акцентним небесно-блакитним кольором, що асоціюється з небом, свободою та подорожами.

Особливістю дизайну є легка стилізація під IT-тематику (назва "SkyBook" — гра слів: sky + book, елементи у вигляді технологічних рішень), що робить бренд упізнаваним та сучасним.

### 🎨 Кольорова палітра:
- **Основний:** #2196F3 (небесно-блакитний — небо та свобода)
- **Вторинний:** #1A237E (глибокий синій — надійність та довіра)
- **Акцентний:** #FF6F00 (помаранчевий — заклик до дії)
- **Успіх:** #4CAF50 (зелений — підтвердження)
- **Фон:** #F5F7FA (світло-сірий — чистота та простір)

## Технологічний стек
Під капотом проєкту працюють сучасні вебтехнології, що забезпечують надійність та швидкість:

### Frontend:
* **Vanilla JavaScript** — чистий код без фреймворків для максимальної продуктивності
* **Fetch API** — асинхронні запити до сервера без перезавантаження сторінки
* **CSS3 Grid/Flexbox** — адаптивна верстка для всіх пристроїв
* **LocalStorage API** — кешування даних для офлайн-доступу
* **Intl.DateTimeFormat** — локалізація дат та часу

### Backend:
* **Node.js** — легкий та швидкий сервер
* **REST API** — стандартизований інтерфейс взаємодії
* **JSON** — зберігання даних у структурованому форматі
* **JWT** — автентифікація користувачів (план)
* **CORS** — безпечна міждоменна взаємодія
