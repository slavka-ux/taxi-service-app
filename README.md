## Про проєкт
QuickRide — це сучасний та швидкий вебдодаток для зручного замовлення таксі онлайн. Головна мета проєкту — надати користувачам максимально простий, інтуїтивно зрозумілий та естетичний інтерфейс для пошуку та замовлення поїздок без зайвого візуального шуму.

Додаток розроблено з використанням чистого JavaScript (Vanilla JS), що забезпечує миттєву роботу без перезавантаження сторінки та мінімальне навантаження на браузер, створюючи відчуття користування нативним застосунком.

## Можливості для користувачів

### 🧑‍💼 Для клієнтів:
* **Швидке замовлення:** Користувачі можуть замовити таксі за лічені секунди, вказавши лише адресу подачі та призначення. Система автоматично розраховує вартість поїздки.
* **Відстеження в реальному часі:** Зручна система статусів дозволяє в реальному часі відстежувати стан замовлення — від пошуку водія до завершення поїздки.
* **Історія поїздок:** Всі замовлення зберігаються в особистому профілі, дозволяючи переглядати історію, повторювати маршрути та аналізувати витрати.
* **Розумний розрахунок:** Система автоматично розраховує орієнтовну вартість поїздки на основі відстані та базового тарифу.

### 🚗 Для водіїв:
* **Доступні замовлення:** Водії бачать список активних замовлень, які очікують на прийняття.
* **Управління статусом:** Можливість змінювати статус поїздки (прийнято, в дорозі, завершено) в один клік.
* **Фільтрація замовлень:** Сортування замовлень за статусом, відстанню або вартістю для зручного вибору.

### 📊 Статуси замовлень:
| Статус | Опис |
|--------|------|
| 🟡 `pending` | Очікує водія |
| 🟢 `accepted` | Водій призначений |
| 🔵 `in-progress` | В дорозі |
| ✅ `completed` | Завершено |
| ❌ `cancelled` | Скасовано |

## Дизайн та інтерфейс
Візуальна складова QuickRide базується на принципах мінімалізму та швидкості. Використано світлу тему оформлення з акцентним жовтим кольором, що асоціюється з таксі та швидкістю пересування.

Особливістю дизайну є легка стилізація під IT-тематику (назва "QuickRide", елементи у вигляді технологічних рішень), що робить бренд упізнаваним та сучасним.

## Технологічний стек
Незважаючи на легкість інтерфейсу, під капотом проєкту працюють сучасні вебтехнології:

### Frontend:
* **Vanilla JavaScript** — для побудови інтерактивних компонентів без зайвих залежностей
* **Fetch API** — для асинхронної комунікації з сервером
* **CSS3 Grid/Flexbox** — для створення адаптивного дизайну
* **LocalStorage API** — для збереження користувацьких налаштувань

### Backend:
* **Node.js** — для швидкого та легкого сервера
* **REST API** — для стандартизованої взаємодії клієнт-сервер
* **JSON** — для зберігання та обміну даними
* **HTTP модуль** — для створення сервера без фреймворків
