## 🎯 Функціональність

### Для клієнтів:
- 📝 Створення замовлення таксі
- 📍 Вибір адреси подачі та призначення
- 💰 Автоматичний розрахунок вартості поїздки
- 📊 Відстеження статусу замовлення
- 📞 Вказування контактних даних

### Для водіїв:
- 📋 Перегляд доступних замовлень
- ✅ Прийняття замовлень
- 🔄 Оновлення статусу поїздки
- 📈 Історія виконаних замовлень

### Статуси замовлень:
- 🟡 **pending** - Очікує водія
- 🟢 **accepted** - Водій призначений
- 🔵 **in-progress** - В дорозі
- ✅ **completed** - Завершено
- ❌ **cancelled** - Скасовано

## 💻 Технології

### Backend:
- **Node.js** - серверна платформа
- **Vanilla JavaScript** - чистий JavaScript без фреймворків
- **HTTP модуль** - створення сервера
- **JSON** - зберігання даних

### Frontend:
- **HTML5** - структура сторінок
- **CSS3** - стилізація інтерфейсу
- **Vanilla JavaScript** - клієнтська логіка
- **Fetch API** - HTTP запити до сервера
